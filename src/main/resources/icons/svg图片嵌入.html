四种嵌入方式对比

1 Base64 嵌入（推荐）

使用 <image href="data:image/png;base64,..."> 将图片编码后直接写入 SVG，单文件，无外部依赖，兼容性最好，适合嵌入网页、邮件签名、图标系统。

2 Data URI 方式

与 Base64 类似但使用 data:image/png;base64,... 完整 URI 格式，某些旧工具（如 Illustrator）识别更好，本质相同，体积略大。

3 foreignObject 方式

通过 <foreignObject> 在 SVG 中嵌入 HTML <img> 标签，灵活性高，但部分环境（如 img 标签内嵌 SVG、PDF 导出）不支持。

4 纯 SVG 路径

仅适用于上传本身就是 .svg 文件的情况，直接将 SVG 内部元素提取出来，体积最小、缩放无损，但位图（PNG/JPG）无法使用此方式。
