## 1. 产品概述

### 1.1 产品定位

CC Assistant 是一款为 JetBrains IDEA 设计的**AI 编程助手插件**，通过集成 Claude Agent SDK，为开发者提供智能对话、代码分析、自动提交等能力。

### 1.2 核心价值

| 价值点 | 描述 |
|--------|------|
| **Claude 集成** | Claude Agent SDK 深度集成，支持多模型切换 |
| **轻量集成** | 基于 Agent SDK Daemon 模式，零冷启动延迟 |
| **原生体验** | 深度融合 IntelliJ Platform，无缝融入开发流程 |
| **美观交互** | 参考 Cursor 设计风格，提供现代化暗色主题 |
| **功能完整** | 支持多会话、流式输出、Token统计、指令集成、MCP |

### 1.3 目标用户

- 已安装 Claude Code CLI 的开发者
- 使用 JetBrains IDE 的程序员
- 需要AI辅助编程的团队
- 需要多模型切换的高级用户

### 1.4 轻量化约束

| 指标 | 目标值     | 说明 |
|------|---------|-----|
| 插件包大小 | < 100MB | 含 ai-bridge 模块 |
| 启动时间增量 | < 500ms | Daemon 常驻，无冷启动 |
| 内存占用 | < 300MB | 正常使用状态 |
| 外部依赖数 | 5       |
| 冷启动响应 | < 500ms | Daemon 已运行时 |

## 3. UI设计规范

### 3.1 设计风格

参考 Cursor IDE 和用户提供的 UI 参考图，采用 **现代暗色极简风格**。

#### 3.1.1 设计原则

| 原则 | 描述 | 实践 |
|------|------|------|
| **极简主义** | 减少视觉噪音，聚焦核心功能 | 精简元素、留白充足 |
| **一致性** | 统一的视觉语言和交互模式 | 组件复用、样式统一 |
| **可访问性** | 高对比度、清晰可读 | 文字大小、颜色对比 |
| **响应式** | 流畅的动画和即时反馈 | 状态过渡、加载指示 |
| **原生感** | 与 IDE 风格融合 | 跟随 IDE 主题、字体 |

#### 6.3.2 JavaScript → Java 通信

```typescript
/**
 * JCEF 桥接服务 (前端)
 */
export class JcefBridge {
  private copyHandler: any = null;
  private messageHandler: any = null;
  
  constructor() {
    this.initializeBridges();
  }
  
  /**
   * 初始化 JCEF 桥接
   */
  private initializeBridges() {
    if (typeof window === 'undefined') return;
    
    const win = window as any;
    
    // 绑定复制处理器
    if (win.CefSharp) {
      this.copyHandler = win.CefSharp.BindObjectAsync('copyHandler');
    }
    
    // 绑定消息处理器 (如果存在)
    this.messageHandler = win.sendMessageHandler;
  }
  
  /**
   * 复制文本到剪贴板
   */
  async copyToClipboard(text: string): Promise<void> {
    if (this.copyHandler) {
      await this.copyHandler.copyToClipboard(text);
    } else {
      // 降级方案: 使用 Clipboard API
      await navigator.clipboard.writeText(text);
    }
  }
  
  /**
   * 发送用户消息
   */
  sendMessage(content: string, attachments?: Attachment[]): void {
    if (this.messageHandler) {
      this.messageHandler.postMessage(JSON.stringify({
        content,
        attachments,
        timestamp: Date.now()
      }));
    }
  }
  
  /**
   * 请求回溯 (Rewind)
   */
  async rewind(sessionId: string, messageId: string): Promise<void> {
    const win = window as any;
    if (win.rewindHandler) {
      await win.rewindHandler.rewind(sessionId, messageId);
    }
  }
  
  /**
   * 请求权限确认 (Plan 模式)
   */
  async requestPermission(toolName: string, toolInput: any): Promise<boolean> {
    const win = window as any;
    if (win.permissionHandler) {
      return await win.permissionHandler.request(toolName, toolInput);
    }
    return false;
  }
}

export const jcefBridge = new JcefBridge();

class JcefBridgeBindings {
    
    private val copyQuery: JBCefJSQuery
    private val messageQuery: JBCefJSQuery
    private val rewindQuery: JBCefJSQuery
    private val permissionQuery: JBCefJSQuery
    
    init {
        // 创建 JBCefJSQuery 实例
        copyQuery = JBCefJSQuery.create(browser)
        messageQuery = JBCefJSQuery.create(browser)
        rewindQuery = JBCefJSQuery.create(browser)
        permissionQuery = JBCefJSQuery.create(browser)
        
        // 绑定复制处理器
        copyQuery.addHandler { json ->
            val data = gson.fromJson(json, CopyData::class.java)
            CopyPasteManager.getInstance().setContents(StringSelection(data.text))
            null
        }
        
        // 绑定消息处理器
        messageQuery.addHandler { json ->
            val data = gson.fromJson(json, MessageData::class.java)
            CliBridgeService.getInstance().sendUserMessage(data.content)
            null
        }
        
        // 绑定回溯处理器
        rewindQuery.addHandler { json ->
            val data = gson.fromJson(json, RewindData::class.java)
            SessionService.rewind(data.sessionId, data.messageId)
            null
        }
        
        // 绑定权限处理器
        permissionQuery.addHandler { json ->
            val data = gson.fromJson(json, PermissionData::class.java)
            // 显示权限确认弹窗
            PermissionDialog.show(data.toolName, data.toolInput) { approved ->
                // 返回批准结果
            }
            null
        }
    }
    
    /**
     * 注入 JS 桥接到页面
     */
    fun injectBridges() {

            window.CefSharp = {

### 6.4 JCEF 约束与最佳实践

> **基于 IntelliJ IDEA 2025.2 + JCEF 实际测试总结 (2026-04-16 新增)**

#### 6.4.1 已验证可行的功能

| 功能 | 状态 | 说明 |
|------|------|------|
| HTML 渲染 | ✅ 可用 | `loadHTML()` 直接加载字符串 |
| CSS 样式 | ✅ 可用 | 渐变、flex、grid 均正常 |
| 基础 DOM 操作 | ✅ 可用 | `getElementById`、`textContent` 正常 |
| 内联事件处理 | ✅ 可用 | `onclick=""` 可以工作 |
| 颜色样式切换 | ✅ 可用 | `style.background` 修改生效 |

#### 6.4.2 已知限制与解决方案

| 限制 | 影响 | 解决方案 |
|------|------|----------|
| `alert()` 被阻止 | 弹窗不显示 | 使用自定义 DOM 元素显示消息 |
| `console.log` 静默 | 控制台不可见 | 桥接到 Java 日志系统 |
| 事件处理器可能被重置 | 按钮只能点击一次 | 使用事件委托或持久化绑定 |
| `window.open` 被阻止 | 无法打开新窗口 | 在 JCEF 内导航或通知 Java |
| LocalStorage 限制 | 持久化可能失效 | 使用 Java 侧存储 |
| ClassLoader 资源路径 | `getResource()` 不稳定 | 使用 `loadHTML()` 或 Base64 |
| 相对路径资源 | JAR 内相对路径可能失效 | 使用绝对 URL 或内联 |

#### 6.4.3 最佳实践

**HTML 结构**
```html

  <meta charset="UTF-8">
  <style>/* 所有 CSS 内联 */</style>
</head>
<body>
  <script>/* 所有 JS 内联 */</script>
</body>
</html>

<!-- ❌ 避免：外部资源 -->
<link href="./style.css">  <!-- 可能加载失败 -->
<script src="./app.js"></script>  <!-- 可能加载失败 -->

### 6.2 Daemon 进程管理

```kotlin
/**
 * Daemon 桥接服务
 */
@Service(Service.Level.APPLICATION)
class DaemonBridgeService : Disposable {
    
    private var daemonProcess: Process? = null
    private val messageCallbacks = ConcurrentLinkedQueue<MessageCallback>()
    private val parser = NDJSONParser()
    
    companion object {
        fun getInstance(): DaemonBridgeService = 
            ApplicationManager.getApplication().getService(DaemonBridgeService::class.java)
    }
    
    /**
     * 启动 Daemon
     */
    fun startDaemon(): Boolean {
        if (daemonProcess?.isAlive == true) return true
        
        val daemonPath = getDaemonPath()
        val nodePath = findNodePath()
        
        val command = listOf(
            nodePath,
            daemonPath,
            "--port", "0"  // 使用 stdin/stdout
        )
        
        val builder = ProcessBuilder(command)
        builder.redirectErrorStream(false)
        builder.environment()["NODE_OPTIONS"] = "--max-old-space-size=512"
        
        daemonProcess = builder.start()
        
        // 启动输出读取线程
        Thread { readDaemonOutput() }.start()
        Thread { readDaemonError() }.start()
        
        // 等待就绪
        return waitForReady(5000)
    }
    
    /**
     * 发送消息
     */
    fun sendMessage(request: QueryRequest): CompletableFuture<QueryResult> {
        val future = CompletableFuture<QueryResult>()
        
        val callback = object : MessageCallback {
            override fun onChunk(chunk: String) {
                // 发布到 UI
                ApplicationManager.getApplication().messageBus
                    .syncPublisher(MessageTopics.CHUNK_RECEIVED)
                    .onChunkReceived(chunk)
            }
            
            override fun onThinking(content: String) {
                ApplicationManager.getApplication().messageBus
                    .syncPublisher(MessageTopics.THINKING_RECEIVED)
                    .onThinkingReceived(content)
            }
            
            override fun onToolUse(name: String, input: JsonObject) {
                ApplicationManager.getApplication().messageBus
                    .syncPublisher(MessageTopics.TOOL_USE_RECEIVED)
                    .onToolUseReceived(name, input)
            }
            
            override fun onComplete(result: QueryResult) {
                future.complete(result)
            }
            
            override fun onError(error: Throwable) {
                future.completeExceptionally(error)
            }
        }
        
        messageCallbacks.add(callback)
        writeToDaemon(serializeRequest(request, callback))
        
        return future
    }
    
    /**
     * 中断当前操作
     */
    fun interrupt() {
        writeToDaemon("{\"type\":\"interrupt\"}\n")
    }
    
    /**
     * 读取 Daemon 输出
     */
    private fun readDaemonOutput() {
        val reader = daemonProcess?.inputStream?.bufferedReader() ?: return
        var line: String?
        
        while (reader.readLine().also { line = it } != null) {
            parser.parseLine(line!!).forEach { message ->
                handleMessage(message)
            }
        }
    }
    
    /**
     * 处理消息
     */
    private fun handleMessage(message: DaemonMessage) {
        when (message.type) {
            "chunk" -> messageCallbacks.forEach { it.onChunk(message.content) }
            "thinking" -> messageCallbacks.forEach { it.onThinking(message.content) }
            "tool_use" -> messageCallbacks.forEach { it.onToolUse(message.toolName, message.toolInput) }
            "complete" -> {
                messageCallbacks.forEach { it.onComplete(message.result) }
                messageCallbacks.clear()
            }
            "error" -> {
                messageCallbacks.forEach { it.onError(Exception(message.error)) }
                messageCallbacks.clear()
            }
        }
    }
    
    override fun dispose() {
        daemonProcess?.destroy()
        daemonProcess = null
    }
}
```

 * 输入循环
 */
function startInputLoop() {
    const rl = readline.createInterface({
        input: process.stdin,
        output: process.stdout,
        terminal: false
    });
    
    rl.on('line', async (line) => {
        try {
            const request = JSON.parse(line);
            await handleRequest(request);
        } catch (error) {
            output({ type: 'error', message: error.message });
        }
    });
    
    rl.on('close', () => {
        process.exit(0);
    });
}

 * 处理请求
 */
async function handleRequest(request) {
    switch (request.type) {
        case 'query':
            await handleQuery(request);
            break;
        case 'interrupt':
            handleInterrupt();
            break;
        case 'switch_model':
            await handleSwitchModel(request);
            break;
        case 'switch_provider':
            await handleSwitchProvider(request);
            break;
    }
}

/**
 * 处理查询请求
 */
async function handleQuery(request) {

    try {

        const runtime = await queryService.getRuntime(sessionId);
        
        // 执行查询

            messages: messages,
            ...options,
            onChunk: (chunk) => {
                output({ type: 'chunk', content: chunk });

            onThinking: (content) => {
                output({ type: 'thinking', content: content });

            onToolUse: (name, input) => {
                output({ type: 'tool_use', toolName: name, toolInput: input });
            }

            type: 'complete',
            result: {
                content: result.content,
                usage: result.usage,
                stopReason: result.stopReason
            }
        });
    } catch (error) {
        output({ type: 'error', message: error.message });
    }

/**
 * Provider 服务
 */
@Service(Service.Level.APPLICATION)
class ProviderService {
    
    private val providers = mutableMapOf<String, ProviderConfig>()
    private var activeProvider: String = "claude"
    
    companion object {
        fun getInstance(): ProviderService = 
            ApplicationManager.getApplication().getService(ProviderService::class.java)
    }
    
    /**
     * 获取活跃 Provider
     */
    fun getActiveProvider(): ProviderConfig {
        return providers[activeProvider] ?: getDefaultClaudeConfig()
    }
    
    /**
     * 切换 Provider
     */
    fun switchProvider(providerId: String) {
        if (providers.containsKey(providerId)) {
            activeProvider = providerId
            
            // 通知 Daemon 切换
            DaemonBridgeService.getInstance().sendCommand(
                mapOf(
                    "type" to "switch_provider",
                    "provider" to providerId,
                    "config" to providers[providerId]
                )
            )
            
            // 发布事件
            ApplicationManager.getApplication().messageBus
                .syncPublisher(ProviderTopics.PROVIDER_CHANGED)
                .onProviderChanged(providerId)
        }
    }
    
    /**
     * 获取可用模型列表
     */
    fun getAvailableModels(providerId: String): List<ModelInfo> {
        return when (providerId) {
            "claude" -> listOf(
                ModelInfo("claude-opus-4-20250514", "Claude Opus 4", Provider.CLAUDE),
                ModelInfo("claude-sonnet-4-20250514", "Claude Sonnet 4", Provider.CLAUDE),
                ModelInfo("claude-3-5-haiku-20241022", "Claude Haiku 3.5", Provider.CLAUDE)
            )
            else -> emptyList()
        }
    }
}

     * 执行 Agent 任务

    suspend fun executeAgent(
        agent: Agent,
        context: ConversationContext,
        onProgress: (AgentProgress) -> Unit
    ): AgentResult = withContext(Dispatchers.IO) {
        
        val result = AgentResult()
        
        try {
            // 1. 准备上下文
            val preparedContext = prepareContext(agent, context)
            
            // 2. 执行主任务
            onProgress(AgentProgress.Stage("主任务执行中", 0))
            
            val mainResult = DaemonBridgeService.getInstance()
                .sendMessage(buildQueryRequest(agent, preparedContext))
                .await()
            
            result.addMessage(mainResult.message)
            
            // 3. 处理子 Agent
            if (mainResult.requiresSubAgent) {
                for (subAgentId in agent.subAgents) {
                    val subAgent = agents[subAgentId] ?: continue
                    
                    onProgress(AgentProgress.SubAgent(
                        subAgentId = subAgentId,
                        status = "运行中",
                        progress = 0
                    ))
                    
                    val subResult = executeSubAgent(subAgent, context, onProgress)
                    result.addSubAgentResult(subAgentId, subResult)
                }
            }
            
            // 4. 收集结果
            result.status = AgentResult.Status.SUCCESS
            
        } catch (e: Exception) {
            result.status = AgentResult.Status.ERROR
            result.error = e.message
        }
        
        result
    }
    
    /**
     * 导出 Agent 配置
     */
    fun exportAgent(agentId: String): String? {
        return agents[agentId]?.let { agent ->
            Json.encodeToString(AgentConfig.serializer(), AgentConfig(

                systemPrompt = agent.systemPrompt,
                model = agent.model,
                permissionMode = agent.permissionMode,
                tools = agent.tools,
                subAgents = agent.subAgents
            ))

        } catch (e: Exception) {

        }
    }

@Service(Service.Level.APPLICATION)
class UsageService {

    private val dailyUsages = mutableMapOf<LocalDate, DailyUsage>()
    
    companion object {
        fun getInstance(): UsageService = 
            ApplicationManager.getApplication().getService(UsageService::class.java)
    }
    
    /**
     * 记录用量
     */
    fun recordUsage(sessionId: String, usage: Usage, model: String, provider: String) {
        // 会话级统计
        val sessionUsage = sessionUsages.getOrPut(sessionId) { SessionUsage(sessionId) }
        sessionUsage.addUsage(usage, model, provider)
        
        // 日级统计
        val today = LocalDate.now()
        val dailyUsage = dailyUsages.getOrPut(today) { DailyUsage(today) }
        dailyUsage.addUsage(usage, model, provider)
        
        // 发布用量更新事件
        ApplicationManager.getApplication().messageBus
            .syncPublisher(UsageTopics.USAGE_UPDATED)
            .onUsageUpdated(sessionId, sessionUsage)
        
        // 检查预算告警
        checkBudgetAlert(dailyUsage)
    }
    
    /**
     * 计算成本
     */
    fun calculateCost(usage: Usage, model: String, provider: ProviderType): Double {
        val pricing = when (provider) {
            ProviderType.CLAUDE -> ClaudePricing.getPricing(model)
            else -> Pricing.default
        }

        val inputCost = (usage.inputTokens / 1_000_000.0) * pricing.inputPrice
        val outputCost = (usage.outputTokens / 1_000_000.0) * pricing.outputPrice
        val cacheWriteCost = (usage.cacheCreationTokens / 1_000_000.0) * pricing.cacheWritePrice
        val cacheReadCost = (usage.cacheReadTokens / 1_000_000.0) * pricing.cacheReadPrice

        return inputCost + outputCost + cacheWriteCost + cacheReadCost
    }
    
    /**
     * 获取统计报告
     */
    fun getUsageReport(days: Int = 7): UsageReport {
        val history = (0 until days).mapNotNull { daysAgo ->
            dailyUsages[LocalDate.now().minusDays(daysAgo.toLong())]
        }
        
        return UsageReport(
            totalTokens = history.sumOf { it.totalTokens },
            totalCost = history.sumOf { it.totalCost },
            dailyBreakdown = history,
            modelBreakdown = history.flatMap { it.modelBreakdown.entries }
                .groupBy { it.key }
                .mapValues { entry -> entry.value.sumOf { it.value } }
        )
    }
}

/**
 * Commit 生成服务
 */
class CommitGenerator(private val project: Project) {
    
    private val git = GitUtil.getGitRepository(project)
    
    /**
     * 生成 Commit 消息
     */
    suspend fun generateCommitMessage(): Result<CommitMessage> = withContext(Dispatchers.IO) {
        try {
            // 1. 获取暂存文件
            val stagedFiles = getStagedFiles()
            if (stagedFiles.isEmpty()) {
                return@withContext Result.failure(NoStagedFilesException())
            }
            
            // 2. 获取 Diff
            val diff = getStagedDiff()
            
            // 3. 分析变更
            val analysis = analyzeChanges(diff, stagedFiles)
            
            // 4. 构建 Prompt
            val prompt = buildCommitPrompt(analysis)
            
            // 5. 调用 AI
            val response = DaemonBridgeService.getInstance()
                .sendMessage(buildQueryRequest(prompt))
                .await()
            
            // 6. 解析结果
            val commitMessage = parseCommitMessage(response.message)
            
            Result.success(commitMessage)
        } catch (e: Exception) {
            Result.failure(e)
        }
    }
    
    /**
     * 分析变更
     */
    private fun analyzeChanges(diff: String, files: List<VirtualFile>): ChangeAnalysis {
        return ChangeAnalysis(
            filesChanged = files.map { it.path },
            additions = countAdditions(diff),
            deletions = countDeletions(diff),
            changeTypes = detectChangeTypes(diff),
            affectedModules = detectAffectedModules(files),
            summary = generateDiffSummary(diff)
        )
    }
    
    /**
     * 构建 Prompt
     */
    private fun buildCommitPrompt(analysis: ChangeAnalysis): String {
        val template = ConfigService.getInstance().getState().commitTemplate
        
        return """
            Based on the following git diff, generate a commit message following ${template} format.
            
            Changed files: ${analysis.filesChanged.joinToString(", ")}
            Change types: ${analysis.changeTypes.joinToString(", ")}
            Additions: ${analysis.additions}, Deletions: ${analysis.deletions}
            
            Diff:
            ```
            ${analysis.summary.take(8000)}
            ```
            
            Generate a commit message with:
            1. A short summary (50 chars max) following ${template} format
            2. Optional body explaining the change
            3. Breaking changes if any
        """.trimIndent()
    }
    
    /**
     * 解析 Commit 消息
     */
    private fun parseCommitMessage(response: String): CommitMessage {
        val lines = response.lines()
        val subject = lines.firstOrNull()?.removePrefix("```")?.trim() ?: ""
        val body = lines.drop(2)
            .takeWhile { !it.startsWith("```") }
            .joinToString("\n")
            .trim()
        
        return CommitMessage(
            subject = subject,
            body = body,
            format = detectCommitFormat(subject)
        )
    }
}
