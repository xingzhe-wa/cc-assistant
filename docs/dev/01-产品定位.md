## 2. 核心价值主张

### 2.1 一句话描述

CC Assistant 是一款深度集成 Claude Agent SDK 的 IntelliJ IDEA 插件，为开发者提供**零延迟启动**、**原生IDE体验**的 AI 编程助手。

### 3.3 用户使用场景

#### 场景 1: 代码审查
```
用户在 IDE 中选中一段代码
    ↓
按 Ctrl+Shift+Q (Quick Fix)
    ↓
CC Assistant 显示代码问题
    ↓
用户接受建议并应用修改
```

#### 场景 2: 上下文对话
```
用户在输入框输入 "帮我分析 @UserService.kt 的性能问题"
    ↓
CC Assistant 自动读取文件内容
    ↓
Claude 分析代码并给出建议
    ↓
用户可以直接在代码块中应用修改
```

#### 场景 3: 自动提交
```
用户完成代码修改，打开 Git 提交对话框
    ↓
CC Assistant 自动分析 Diff
    ↓
显示 AI 生成的提交信息
    ↓
用户确认后提交
```

## 5. 产品差异化

### 5.1 竞品对比

| 功能特性 | CC Assistant | Cursor IDE | GitHub Copilot | Codeium |
|---------|-------------|-----------|----------------|---------|
| Claude 集成 | ✅ 原生 | ✅ 内置 | ❌ | ❌ |
| IDE 集成度 | ⭐⭐⭐⭐⭐ | N/A | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| 冷启动速度 | < 500ms | N/A | ~2s | ~1s |
| @file 引用 | ✅ | ✅ | ❌ | ✅ |
| Diff 审查 | ✅ | ✅ | ❌ | ❌ |
| 多 Provider | ✅ | ❌ | ❌ | ❌ |
| Agent 系统 | ✅ | ❌ | ❌ | ❌ |
| MCP 支持 | ✅ | ❌ | ❌ | ❌ |
| 权限模式 | ✅ 3 种 | ❌ | ❌ | ❌ |
| 开源 | ✅ 计划中 | ❌ | ❌ | ✅ |

### 5.2 核心差异化点

#### 1. Claude 原生深度集成
- **优势**: 基于 Claude Agent SDK，支持最新 Claude 模型
- **竞品**: Copilot 仅支持 OpenAI 模型，其他竞品无 Claude 深度集成

#### 2. Daemon 零延迟架构
- **优势**: 常驻进程架构，响应时间 < 500ms
- **竞品**: 需要冷启动，响应时间 > 2s

#### 3. 完整的 Agent 系统
- **优势**: 支持自定义 Agent、子 Agent 追踪、Skills 指令
- **竞品**: 无类似功能

#### 4. 三级权限模式
- **优势**: Default/Sandbox/YOLO 三种模式，适应不同场景
- **竞品**: 无权限控制或仅有单一模式

#### 5. 轻量化设计
- **优势**: 插件包 < 100MB，仅依赖 Gson
- **竞品**: 依赖复杂，包体积大
