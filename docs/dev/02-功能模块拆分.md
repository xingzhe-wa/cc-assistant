## 2. 表现层模块 (UI Layer)

### 2.1 Chat Panel 模块

**模块职责**: 主对话界面，包含所有会话相关 UI

#### 组件拆分

| 组件名称 | 文件路径 | 职责描述 | 技术方案 |
|---------|---------|---------|---------|
| **ChatPanel** | `ui/chat/ChatPanel.kt` | 主面板容器，协调子组件 | `JPanel` + `BorderLayout` |
| **HeaderPanel** | `ui/chat/HeaderPanel.kt` | 顶部标题栏，Provider 状态显示 | `JPanel` + `FlowLayout` |
| **SessionListPanel** | `ui/chat/SessionListPanel.kt` | 会话列表，支持搜索、收藏 | `JList` + 自定义 `ListCellRenderer` |

## 5. 数据模型模块 (Model Layer)

### 5.1 会话模型

```kotlin
// 会话数据结构
data class ChatSession(
    val id: String,
    val createdAt: Instant,
    var updatedAt: Instant = createdAt,
    var title: String? = null,
    var messages: MutableList<Message> = mutableListOf(),
    var rewindPoints: MutableList<RewindPoint> = mutableListOf(),
    var isFavorite: Boolean = false,
    var context: ConversationContext = ConversationContext(),
    var provider: String = "claude",
    var model: String = "claude-sonnet-4-20250514"
)

// 消息数据结构
data class Message(
    val id: String,
    val role: Role,
    val content: String,
    val timestamp: Instant,
    val usage: Usage? = null,
    val attachments: List<Attachment> = emptyList(),
    val fileReferences: List<FileReference> = emptyList(),
    val thinkingContent: String? = null,
    val toolCalls: List<ToolCall> = emptyList()
)

enum class Role { USER, ASSISTANT, SYSTEM, TOOL }
```

### 5.2 Agent 模型

```kotlin
data class Agent(
    val id: String,

    val description: String,
    val systemPrompt: String,
    val model: String,
    val permissionMode: PermissionMode,
    val tools: List<String>,
    val subAgents: List<String>
)

enum class PermissionMode { DEFAULT, SANDBOX, YOLO }
```
